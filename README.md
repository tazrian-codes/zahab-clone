# 🛍️ Zahab Perfume Clone (Frontend)

A modern e-commerce perfume website clone built using React. This project focuses on creating a smooth and interactive shopping experience with essential e-commerce functionalities and a responsive user interface.

---

## 🚀 Features

- 🛒 Add to Cart (with size selection)
- ➕➖ Quantity selection before adding to cart
- ❤️ Wishlist system
- 🔍 Live search with suggestions
- 👀 Quick view (product glimpse modal)
- 🧾 Checkout page with dynamic order summary
- 📦 Cart & Wishlist persistence using Local Storage
- 📱 Responsive design for both desktop and mobile devices
- 🔄 Product sorting (price, alphabetical)
- 🧩 Category-based product organization

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Context API (State Management)
- CSS (Custom styling)
- FontAwesome Icons

---

## 📂 Project Structure

src/
 ├── components/
 ├── pages/

---

## 🧠 Key Concepts Implemented

- Global state management using Context API  
- Dynamic cart system (product + size based key)  
- Conditional rendering  
- Event handling & propagation control  
- Local storage for persistent data  
- Component-based architecture  

---

## 🚧 Future Improvements

Most of the core e-commerce features like cart management, sorting, category-based browsing, and search have already been implemented. However, there is still room for further refinement and enhancement in terms of UI/UX, performance optimization, and additional interactive features to make the application more polished and production-ready.
